Frequenly Asked Questions
  1. When was this built?
    This application was built in March 2010 for submission in the apps4NSW competition.
  2. What is this used for?
    Where4NSW is a web-application for finding areas you might want to live. You can find areas by population, climate, fire danger and much more!
  3. What technologies are used?
    jQuery is used for display/layout. google project hosting is used for subversion repository google app engine is used for deployment
  4. What license?
    GPL version 3
  5. What is the need being served?
    finding suitable neighbourhoods to live
  6. What is the motivation behind the user interface design?
    The application is designed using standards-compliant HTML/CSS for accessibility. All of the functionality is on one page, so navigation is not a problem. No frustrating signup is required.
  7. What data sources were used?
    • NSW state and regional indicators (housing)
    • fire incidents
